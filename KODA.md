# KODA — Конфигурация CLI-агента

## Источники документации

### The Rust Programming Language
Официальная книга по Rust: https://doc.rust-lang.org/book/

#### Структура книги (21 глава + приложения):

**Основы:**
1. Getting Started — установка, Hello World, Cargo
2. Programming a Guessing Game — практический проект
3. Common Programming Concepts — переменные, типы данных, функции, управление потоком
4. Understanding Ownership — владение, ссылки, заимствование, срезы
5. Structs — структуры и методы
6. Enums and Pattern Matching — перечисления и match
7. Packages, Crates, and Modules — модульная система
8. Common Collections — Vec, String, HashMap
9. Error Handling — panic!, Result
10. Generic Types, Traits, and Lifetimes — дженерики, трейты, времени жизни

**Продвинутые темы:**
11. Writing Automated Tests
12. An I/O Project — CLI-приложение
13. Functional Language Features — замыкания, итераторы
14. More about Cargo and Crates.io
15. Smart Pointers — Box, Rc, RefCell
16. Fearless Concurrency — многопоточность
17. Async/Await — асинхронное программирование
18. OOP Features — объектно-ориентированное программирование
19. Patterns and Matching
20. Advanced Features — unsafe Rust, макросы
21. Final Project — веб-сервер

**Приложения:**
- Keywords, Operators, Derivable Traits, Development Tools, Editions, Translations, Nightly Rust
