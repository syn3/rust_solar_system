# SolarApp

A Rust application built with the macroquad framework.

## Description
SolarApp is a 2D game/application developed using Rust and the macroquad library. It provides a platform for creating interactive 2D experiences.

## Features
- Built with Rust for performance and safety
- Uses macroquad for graphics and game development
- Modular architecture with physics and UI components

## Requirements
- Rust 2024 edition
- Cargo package manager

## Installation
1. Make sure you have Rust installed on your system
2. Clone this repository
3. Navigate to the project directory
4. Run `cargo run` to build and run the application

## Usage
Run the application with:
```bash
cargo run
```

## Project Structure
- `src/main.rs`: Main entry point of the application
- `src/physics/`: Physics-related modules
- `src/ui/`: User interface components
